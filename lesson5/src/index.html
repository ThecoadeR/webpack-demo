使用Babel处理es6语法

Babel文档

babel-loader实际上是一个桥梁 并不做翻译功能 webpack和babel的通信桥梁

@babel/preset-env才是真正的翻译器

引入babel/polyfill 通过这个包可以在低版本浏览器里注入一些低版本浏览器所缺失的代码

由于引入了polyfill代码体积会变的非常巨大 需要按需引入 可以做如下配置(新版本里需要安装core-js）

module: { rules: [ { test: /\.js$/, exclude: /node_modules/, loader: 'babel-loader', options: { presets: [['@babel/preset-env', { useBuiltIns: 'usage', // 不添加所有的新特性 只添加用到了的特性 减少打包以后的代码体积 corejs: 3 }]] } } ] }
